# BLR

This is a remake of the http://biolitrepo.org. While it looks almost the same as the original, it has been  completely rewritten in plain html and JavaScript. Unlike the old version, this requires no `webpack`, no `nodejs` to compile and prepare the files, and at 170KB, it is almost two orders of magnitude less in size (< 60 KB vs > 4.5 MB).
